## What is Tig?

Tig is a platform that automatically captures AI conversations from tools like **Cursor**, **Claude Code**, and **Windsurf**, linking them to your commits and pull requests. It provides your team with searchable context for every line of code built with AI assistance.

<CardGroup cols={2}>
<Card title="Automatic Capture" icon="robot">
  Tig runs in the background during git commits and pushes—no manual saving required
</Card>

<Card title="PR View" icon="code-pull-request">

</Card>

## How It Works

<Steps>
<Step title="Install Tig">
  Install the NPM package and authenticate with GitHub

  ```bash
  npm install -g @tigtech/simple-tig
  tig login
  ```
</Step>

<Step title="Set Up Your Project">
  Run setup in your project directory to configure automatic capturing

  ```bash
  cd /path/to/your/project
  tig setup
  ```
</Step>

<Step title="Code as Usual">
  Continue using your AI assistant—Tig automatically captures conversations on commit asynchronously
</Step>

</Steps>
